Principais formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica, que não tem um significado, só forma)

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica, que tem um significado no site, é um termo de destaque)

Itálico/Ênfase

Nessa frase, temos um termo itálico usando a tag I (Não semântica)

Nesta frase, temos um termo em Ênfase usando a tag EM (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK

Para mudar a cor dentro do mark escreva style + background color e escolha a cor

Para mudar a cor de varias marks de uma só vez crie uma tag STYLE / MARK { BACKGROUND-COLOR

Textos Grandes e Pequenos

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Pode ser marcar um texto como excluído/riscado (semântica)

Texto inserido

Marcar como um texto inserido para dar Ênfase e indicar que elee foi add (semântica)

Existe também o sublinhado com a tag U (Não semântica)

Texto sobrescrito

para isso usamos o SUP como: X 20 +3

Texto subscrito

Para inserir coisas como: H 2 O. Usar a tag SUB
